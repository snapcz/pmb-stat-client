# Ini adalah sebuah reminder mengenai cara deploy local Server untuk
# Flask API - API yang digunakan dengan memanfaatkan python
# menggunakan anaconda (beda kalau nge set pake python biasa)

1. open anaconda prompt
2. redir to project location which contain the project.py
	e.g: cd D:/Kuliah/Semester 7/Prosi 2/Test Flask - Project
3. run command: set FLAKS_APP=your_project_name.py
	e.g: set FLASK_APP=flask_app_test.py
4. run command: flask run
